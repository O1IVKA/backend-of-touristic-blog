# Hiker project - server 

## Project setup
```
virtualenv venv
source venv/bin/activate
python -m pip install -r requirements.txt
```

### Stack
- FastAPI users
- Alembic
- SQLalchemy
- Pydantic
- fastapi/django - filters
- FastAPI
- PostgreSQL/sqlite3
- Docker + uvicorn
